# CDNet Tracking Evaluation

This project evaluates classical object tracking algorithms on the CDNet2012 dataset.

Trackers implemented:
- MOSSE
- CSRT
- KCF
- CamShift

Metrics:
- IoU (Intersection over Union)
- CLE (Center Location Error)
- Precision
- Recall
- F1-score

Dataset:
CDNet2012 – highway sequence

Additional comparison:
YOLOv8 + ByteTrack for multi-object tracking.

## How to run
python run_experimentms.py
